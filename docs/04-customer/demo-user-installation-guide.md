# EC Ranger デモインストール手順

## インストール手順（3分で完了）

### 📍 インストールURL
```
https://white-island-08e0a6300.2.azurestaticapps.net/install
```

---

### ステップ1️⃣ URLにアクセス
上記URLをクリックして、インストールページを開いてください。

**⚠️ 注意**: 
- 必ず `/install` で終わるURLをクリックしてください
- トップページ（`/`）は「Shopify認証が必要です」と表示されます（正常です）
- インストール完了後は自動的にアプリが開きます

### ステップ2️⃣ ストアドメインを入力
入力欄に、お客様のShopifyストアドメインを入力してください。

**入力例：**
- `your-store` （.myshopify.comは自動で追加されます）
- または `your-store.myshopify.com` （フルドメインでもOK）

### ステップ3️⃣ 「アプリをインストール」ボタンをクリック
緑色のボタンをクリックすると、Shopifyのログイン画面に移動します。

### ステップ4️⃣ Shopifyにログイン
- Shopifyストアの**管理者アカウント**でログインしてください
- 2段階認証が設定されている場合は、認証コードも入力

### ステップ5️⃣ アプリの権限を承認
権限確認画面が表示されます。

**必要な権限：**
- ✅ 注文情報の読み取り
- ✅ 商品情報の読み取り  
- ✅ 顧客情報の読み取り

「**インストール**」ボタンをクリックして承認してください。

### ステップ6️⃣ インストール完了
EC Rangerの管理画面にリダイレクトされます。

## ⚠️ ご注意事項

### インストール前の確認
- Shopify管理者権限を持つアカウントでログインしてください
- ブラウザのポップアップブロッカーを無効にしてください
- Cookieを有効にしてください

### トラブルシューティング

**Q: インストールボタンを押してもエラーになる**
- ストアドメインが正しいか確認してください
- 一度ブラウザを更新してから再度お試しください

**Q: Shopifyログイン後に元の画面に戻ってしまう**
- Cookieが有効になっているか確認してください
- 別のブラウザ（Chrome推奨）でお試しください

**Q: 権限承認画面が表示されない**
- すでにインストール済みの可能性があります
- Shopify管理画面の「アプリ」から確認してください
