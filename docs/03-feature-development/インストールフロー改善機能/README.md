## 🔍 現状の問題

### 症状
1. ルートページ（`/`）で「認証状態を確認中...」と表示されたまま進まない
2. `Stores`テーブルにレコードが作成されない
3. 401エラーが発生する（`/api/store`、`/api/subscription/status`など）
