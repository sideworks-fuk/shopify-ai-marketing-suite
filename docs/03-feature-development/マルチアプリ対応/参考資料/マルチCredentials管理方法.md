# マルチCredentials管理方法（参考資料）

> **注意**: このドキュメントは旧設計（Stores.ApiKey/ApiSecret活用）の参考資料です。  
> 現在の設計では`ShopifyApps`テーブルを使用します。  
> 最新の設計については `../ShopifyAppsテーブル設計書.md` を参照してください。

---

## 概要

このドキュメントは、マルチアプリ対応の設計変遷を記録するための参考資料です。

### 旧設計（非推奨）

- `Store`テーブルの`ApiKey`/`ApiSecret`フィールドを直接使用
- ストアごとに異なるCredentialsを`Store`テーブルに保存

### 新設計（現在）

- `ShopifyApps`テーブルでアプリ情報を一元管理
- `Store`テーブルに`ShopifyAppId`を設定して関連付け

詳細は `../ShopifyAppsテーブル設計書.md` を参照してください。
