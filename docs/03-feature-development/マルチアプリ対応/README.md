# マルチアプリ対応ドキュメント

このフォルダには、Shopifyアプリのマルチアプリ対応（公開アプリとカスタムアプリの両方に対応）に関する設計書と実装ドキュメントを格納しています。

## ドキュメント一覧

### 設計書

1. **マルチアプリ対応設計書.md**
   - マルチアプリ対応の全体設計
   - データベースベースのCredentials管理
   - 実装ロジックの詳細

2. **マルチアプリ対応アーキテクチャ.md**
   - システムアーキテクチャの概要
   - コンポーネント間の関係

3. **公開アプリとカスタムアプリの構成パターン.md**
   - 3つの構成パターンの比較
   - パターン2（バックエンド共有）の採用理由

4. **ShopifyAppsテーブル設計書.md**
   - ShopifyAppsテーブルの設計
   - エンティティモデル
   - マイグレーション手順

### 実装ドキュメント

5. **実装方針決定書.md**
   - パターン2（バックエンド共有）の採用決定
   - 実装構成とメリット・デメリット

6. **インストール時のAPI Key判定方法.md**
   - 初回インストール時のAPI Key判定ロジック
   - フロントエンドからAPI Keyを渡す方法

7. **実装前確認事項.md**
   - 実装前のチェックリスト
   - 検証項目

### 管理ドキュメント

8. **マルチCredentials管理方法.md**
   - Credentials管理の詳細
   - データベースベースの管理方法

9. **バックログ-マルチアプリ対応.md**
   - タスク管理と進捗状況

10. **公開アプリとカスタムアプリの構成パターン-レビュー.md**
    - レビュー記録

## ドキュメントの読み方

### 初めて読む場合

1. **マルチアプリ対応アーキテクチャ.md** - 全体像を把握
2. **公開アプリとカスタムアプリの構成パターン.md** - 採用パターンの理解
3. **実装方針決定書.md** - 実装方針の確認

### 実装を進める場合

1. **ShopifyAppsテーブル設計書.md** - データベース設計
2. **インストール時のAPI Key判定方法.md** - インストールフローの実装
3. **実装前確認事項.md** - 実装前のチェック

### 詳細を確認する場合

1. **マルチアプリ対応設計書.md** - 詳細設計
2. **マルチCredentials管理方法.md** - Credentials管理の詳細
