# docs/deployment フォルダ整理計画

## 方針

`docs/deployment`フォルダは命名規則に反しているため、以下の方針で整理します：

1. **Azure関連の設定・トラブルシューティング** → `docs/05-development/04-Azure_DevOps/`
2. **データベース関連** → `docs/05-development/03-データベース/マイグレーション/`
3. **デバッグ・トラブルシューティング** → `docs/05-development/08-デバッグ・トラブル/04-troubleshooting/`
4. **Shopify関連** → `docs/05-development/06-Shopify連携/`
5. **環境変数設定** → `docs/05-development/01-環境構築/`
6. **GitHub Actions関連** → `docs/05-development/04-Azure_DevOps/デプロイメント/`
7. **作業ログとして記録** → `docs/worklog/2025/12/`
