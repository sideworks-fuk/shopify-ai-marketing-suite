# サブスクリプション追加依頼ガイド（Amazing Day案件）



## 概要

Amazing Day案件の本番環境構築を代行させていただくため、ジェイドゥーのAzure環境へのアクセス権限をお願いいたします。

**代行作業の目的**: ジェイドゥー Azureアカウントに本番環境を構築し、システム運用を実現する



## 必要なアクセス権限

### 基本権限

- **Azure サブスクリプションの「共同作成者」権限**
  - リソースの作成・変更・削除が可能

## 追加作業依頼

- プロジェクト専用サブスクリプション（必須）

**補足**

「Amazing Day」専用のサブスクリプションプランを「アクセスネット」とは別に分けて管理する。

### 新規サブスクリプション作成手順

#### Step 1: Azureポータルにログイン

1. [https://portal.azure.com](https://portal.azure.com) にアクセス
2. ジェイドゥーの管理者アカウントでログイン

#### Step 2: サブスクリプション作成

1. 「サブスクリプション」を検索・選択
2. 「+ 追加」ボタンをクリック
3. 「サブスクリプション」を選択

#### Step 3: 基本情報の入力

1. **ディレクトリ**: ジェイドゥーのディレクトリを選択
2. **プラン**: 「従量課金制」を選択
   - ⚠️ **重要**: 画面では「Microsoft Azure Plan」という表記は表示されません
   - 「最も人気のあるオファー」セクションの「**従量課金制**」を選択してください
   - 「標準オファーを選択します」ボタンをクリック
   - 実際のプラン名は「Azure プラン」（Azure Plan）となります
   - 「アクセスネット」サブスクリプションと同じプランです
3. **サブスクリプション名**: **「Amazing Day」** を入力
4. **説明**: 「Amazing Day案件用」と記載

#### Step 4: 課金情報の設定

1. **課金アカウント**: ジェイドゥーの課金アカウントを選択
2. **請求先住所**: 正しい請求先住所を確認
3. **支払い方法**: クレジットカードまたは請求書を選択

#### Step 5: サブスクリプションの作成

1. 入力内容を確認
2. 「作成」ボタンをクリック
3. 作成完了まで数分待機

#### Step 6: 作業者への権限付与

1. 作成されたサブスクリプション「Amazing Day」を選択

2. 左メニューから「アクセス制御 (IAM)」を選択

3. 「追加」→「ロールの割り当ての追加」をクリック

4. **ロール**：「特権管理者ロール」タブを選択し、検索バーに「共同作成者」と入力して表示された「共同作成者」を選択

5. **アクセスの割り当て先**：「ユーザー、グループ、またはサービス プリンシパル」

6. **選択**：作業者のメールアドレスを入力

   ```
   作業者メールアドレス: fukuda@jdo.jp
   ```

7. **最終確認**：「レビューと割り当て」ボタンをクリックして権限の割り当てを完了
