## 💰 課金プラン概要

| プラン | 月額 | 無料期間 | 対象 |
|--------|------|---------|------|
| **Starter** | $50 | 7日間 | 小規模店舗（商品1,000点まで） |
| **Professional** | $80 | 7日間 | 中規模店舗（商品10,000点まで） |
| **Enterprise** | $100 | 14日間 | 大規模店舗（無制限） |

### 主な機能差
```
Starter        : 基本分析、月次レポート
Professional   : + 高度な分析、週次レポート、API連携
Enterprise     : + カスタマイズ、専任サポート、無制限API
```
