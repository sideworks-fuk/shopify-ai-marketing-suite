# 申請タスク: API連絡先メールアドレス修正

## 作成日: 2025-12-31
## 優先度: 高
## 状態: 未着手

## 問題概要

Shopify Partners Dashboardの「予備ステップ」で、API連絡先メールアドレスに関するエラーが発生しています。

### エラー内容
- **エラーメッセージ**: 「Shopify」という単語が含まれないAPI連絡先メールアドレスを使用してください
- **現在の設定**: `shopify@access-net.co.jp` ❌
- **エラー理由**: メールアドレスに「Shopify」という単語が含まれているため

## 対応内容

### 1. 新しいメールアドレスの決定

**推奨オプション:**
- `app@access-net.co.jp` ✅（推奨）
- `support@access-net.co.jp` ✅
- `ec-ranger@access-net.co.jp` ✅

**決定事項:**
- [ ] 新しいメールアドレスを決定
- [ ] メールアドレスの作成・設定（必要に応じて）

### 2. Shopify Partners Dashboardでの修正

**修正手順:**
1. [ ] Shopify Partners Dashboardにログイン
   - URL: https://partners.shopify.com
2. [ ] Apps > EC Ranger を選択
3. [ ] Settings を選択
4. [ ] Contact information セクションを開く
5. [ ] API contact email の「Edit」ボタンをクリック
6. [ ] 新しいメールアドレスを入力
7. [ ] 保存をクリック

### 3. 修正後の確認

- [ ] Partners Dashboard > Distribution > 予備ステップ を確認
- [ ] 「設定の要件に関する問題を解決してください」セクションのエラーが解消されていることを確認
- [ ] グレーのチェックマークが緑色のチェックマークに変わっていることを確認

## 注意事項

- メールアドレスを変更しても、既存のメール転送設定があれば影響はありません
- 新しいメールアドレスがShopifyからの通知を受信できることを確認してください
- 変更後は、申請前の最終チェックで再度確認してください
