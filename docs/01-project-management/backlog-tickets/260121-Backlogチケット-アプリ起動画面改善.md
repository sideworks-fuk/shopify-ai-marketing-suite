# Backlogチケット: アプリ起動画面の改善

## タイトル
アプリ起動時の自動接続フロー改善とアイコン差し替え

## 種別
改善

## 優先度
中

## 説明

### 問題1: アプリ起動時の自動接続フロー

**現状**:
- アプリ起動時に必ず「接続」画面が表示される
- ストアドメインの変更もできない状態
- ユーザーが手動で接続ボタンを押す必要がある

**改善内容**:
- 自動で認証まで進んで初期画面に遷移するように改善
- ユーザビリティ向上のため、一気に画面まで進むフローに変更

**注意事項**:
- ややこしそうな場合は後回しの可能性あり（ただし多分すぐできる）

### 問題2: アイコンの差し替え

**現状**:
- 現在のアイコンがECレンジャー用ではない

**改善内容**:
- ECレンジャー用のアイコンに差し替え
- 中川さんからアイコンを受け取っている
