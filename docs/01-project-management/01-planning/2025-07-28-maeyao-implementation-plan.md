### 3.2 顧客セグメント分析
- **常連客**: 月2回以上購入
- **激辛愛好家**: ★4以上を主に購入
- **サブスク会員**: 定期購入顧客
- **早稲田ローカル**: 配送先が新宿区・文京区

### 3.3 季節性分析
- 夏場の激辛カレー需要
- 冬場の温まりカレー需要
- 新商品投入時期の最適化

## 4. 想定される分析ニーズ

### 4.1 売上分析
- 商品タイプ別売上（冷凍 vs レトルト）
- 辛さレベル別売上推移
- 13,000円以上（送料無料ライン）の注文分析

### 4.2 顧客分析
- リピート率の高い商品組み合わせ
- 初回購入から定期購入への転換率
- 休眠顧客の再活性化（2017年閉店時の顧客）

### 4.3 在庫最適化
- 人気商品の在庫切れ防止
- 季節需要予測
- 新商品の需要予測
