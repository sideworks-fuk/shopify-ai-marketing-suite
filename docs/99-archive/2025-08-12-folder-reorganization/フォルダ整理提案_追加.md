### 2. ADRフォルダの統合

現在の状況：
```

└── adr/                 # Architecture Decision Records
    ├── ADR-000-template.md
    └── ADR-001-hangfire-vs-azure-functions.md
```

#### 統合提案

**Option A: 02-architectureに統合（推奨）**
```
docs/02-architecture/
├── 01-システム設計/
├── 02-データベース設計/
├── 03-技術スタック/
```

```
docs/01-project-management/
├── 00_meeting/
├── 01-planning/
├── 04-organization/
└── 05-adr/             # ← 新規作成
```

**理由：**
